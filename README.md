# AMSExpansaoDicionarioLIBRAS
Este é um trabalho final da disciplina de análise e modelagem de sistemas, onde se propõe uma forma de atualizar o dicionário de LIBRAS existente.
Neste projeto, o Trello foi utilizdo como forma d eorganizar as tarefas e o fluxo do desenvolvimento, da mesma forma, a maior parte dos textos e dados
foram organizados utilizando o aplicativo Obsidian(https://obsidian.md/), com o arquivo tela.canvas sendo o centro.
